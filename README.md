# Qubit_Simulation
目標是做給C++使用的量子計算模擬標頭檔  
目前進度已經將量子糾纏以及常見基本量子閘還有量子觀測完成了

我的C++使用環境為Eclipse(2024/12) C++17 gcc (Rev2, Built by MSYS2 project) 14.2.0

---------------------------------------分隔線-------------------------------------  
目前實現功能:  
初始化量子位元  
常見基本量子閘以及部份其他量子閘  
量子觀測  

尚未實現功能:  
自定義量子閘(在如何將輸入矩陣轉換成么正矩陣方面陷入瓶頸)
